# Reference Scan

## Older Morning Tee Practice Prototype

The older prototype is visual and product inspiration only. The current Vite + React app remains the source of truth.

### Worth Keeping

- Compact mobile-shell feel.
- Simple putting entry.
- Green map clarity.
- Puck/island path energy.
- Small reward animation moments.

### Avoid

- Fake leaderboard.
- Profile/settings.
- XP economy.
- Social league.
- Generic badges.
- Hardcoded 5-putt flow.
- Shallow scoring.
- Replacing the current React architecture.

### What Can Inspire The Current App

- Make the current progress path feel more like skill islands than generic rounds.
- Use small pop, sweep, and sparkle moments after a completed practice action.
- Keep logging visually clear: tap cup for makes, tap the green for misses.
- Keep the mobile shell compact so the user can finish quickly outdoors.

### Decision

The current Vite + React app remains the source of truth. The old prototype is only visual/product inspiration.

# Deployment

Morning Tee Practice should be deployed as its own standalone frontend app.

## Vercel Preview Setup

- Vercel project should connect to this repo only: `harrycarlisle/morning-tee-practice`.
- Framework: Vite.
- Build command: `npm run build`.
- Output directory: `dist`.
- Use the Vercel preview URL for testing.
- Do not connect a custom domain yet.

## Domain Guardrail

Do not connect `morningtee.com` or `practice.morningtee.com` until the preview build has been reviewed and approved.
